# api-rest-Desenvolvimento-WEB-BackEnd

Trabalho de Desenvolvimento WEB - Back End (765810) 
O objetivo deste trabalho é implementer uma API RESTful para gerenciamento de
tarefas. A API deverá permitir o cadastro, consulta, atualização e remoção de
tarefas, contendo informações como nome da tarefa, data de entrega e
responsável.
